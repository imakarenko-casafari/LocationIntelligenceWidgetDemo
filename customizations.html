Load on click & preloader

You can run widget on eny JS event after page loaded.
For customization pre loader just add any code into tag wich will be target.

Write here any what u want to ovverride default preloader...
Loading... 


<script type="text/javascript" src="https://casafari.com/location-intelligence/widget.js"></script>
<div id="LocationIntelligenceWidget1">
    Write here any what u want to ovverride default preloader...
    <div class="spinner-border text-primary" role="status">
        <span class="visually-hidden">Loading...</span>
    </div>
    <button type="button" class="btn btn-primary" id="render">Render Widget</button>
</div>
<script type="text/javascript">
    document.getElementById('render').onclick = () => window.Casafari.locationIntelligenceWidget({
        "key": "{{YOUR_WIDGET_KEY}}",
        "lat": 39.7222524,
        "lng": -8.1393366,
        "lang": "it",
        "targetID": "LocationIntelligenceWidget1",
    });
</script>
<style>
    #LocationIntelligenceWidget1 {
        display: flex;
        flex-direction: column;
        align-content: space-around;
        align-items: center;
        gap: 15px;
    }
</style>
