Handle Widget Events

What u need just add JS to handle it. available events

  • CASAFARI_WIDGET_EVENT_RESIZE - on resize Widget.
    *Default: change iframe height, to get rid of the scroll.
    *Example: when change mobile/desktop view. 

<script type="text/javascript" src="https://casafari.local/location-intelligence/widget.js"></script>
<div id="LocationIntelligenceWidget"></div>
<script type="text/javascript">
    window.Casafari.locationIntelligenceWidget({
        "key": "{{YOUR_WIDGET_KEY}}",
        "lat": 38.7222524,
        "lng": -9.1393366,
    });
    window.addEventListener('message', (event) => {
        console.log('Handle Widget's Event', event);
        alert(JSON.stringify(event.data));
        if (event.data.type === 'ANY_EVENT') {
            // do smth
        }
    });
</script>

Important

Check html of page where you want to use widget.
Language of widget will be detected from tag: <html lang="en">
Mobile responsibility will work correct with tag: <meta name="viewport" content="..., initial-scale=1" />
