Issue #1: Widget Not Working

Check that domain of website is correct. You should compare domain in settings with referrer in request headers (Browser).

Browser allways is right. Is is auto add this field, and you can't change it.
*Steps to get it: F12 -> Network (tab) -> Select request (it will be subdocument to one of casafari domains).

Should add domains of site where you will use Widget. A valid domain requires a host and must not include any path, port, query or fragment. In this case paste "imakarenko-casafari.github.io".
