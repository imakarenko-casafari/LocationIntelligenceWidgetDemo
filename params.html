Example of all available Widget's params.

Required params mark as `*`, all other are optional:

  • key* - your widget key
  • lat* - latitude of estate
  • lng* - longitude of estate
  • lang - for hard set language of Widget.
    *Default
    *Available
  • hideMarker - hide marker of estate on the map.
    *Default: true
    *Available: false
  • pinRadius - Sets the radius in metters of estate intead marker.
    *Default: 1000
    *Available: 0 - for disable
  • minZoom - Sets the map's minimum zoom level. Other factors such as map height may restrict zooming.
    *Default: -2
  • maxZoom - Sets the map's maximum zoom level. Must be greater than the `minZoom`.
    *Default: 22
  • targetID - ID of tag for render widget. Use it for render more than one widget at the page.
    *Default: `LocationIntelligenceWidget`
  • locationScore - options of Location Score component on the right bottom coner of map. Use for sort and filter categories.
    *Default: `['cafe', 'nightlife', 'restaurant', 'pharmacy', 'groceries', 'shopping']` or `false for disable`
    *Available: array of any categories from default list, as example: `['shopping', 'cafe']` or `false` for disable component.
  • callbackUrl - url for catch POST callback of event, client has submitted contact us form.

    For testing we recommend use Webhook.site, see example.
